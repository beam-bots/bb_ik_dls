A DLS-based inverse kinematics solver for the [Beam Bots](https://github.com/beam-bots/bb) robotics framework.

## Installation

Add `bb_ik_dls` to your list of dependencies in `mix.exs`:

```elixir
def deps do
  [
    {:bb_ik_dls, "~> 0.3.0"}
  ]
end
```
